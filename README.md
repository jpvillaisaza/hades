# Hades

Hades is a collection of games.

## Installation

### With Cabal

```
git clone git@github.com:jpvillaisaza/hades
cd hades
cabal build
cabal install
```

## Usage

```
hades <game>
```

## Games

### Bagels

```
hades bagels
```

### Hangman

```
hades hangman
```

### Word

```
hades word
```
